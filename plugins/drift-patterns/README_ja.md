# drift-patterns plugin

*[English](README.md) | [日本語](README_ja.md)*

`drift-patterns` スキルを提供する: 10 種の静かなエージェント失敗モードと構造的対策を自己完結でまとめたカタログに、診断テーブルと設計時チェックリストを添えたもの。

## インストール

```text
/plugin marketplace add hiro178/agent-harness-lab
/plugin install drift-patterns@agent-harness-lab
```

## 発動タイミング

以下の場面でオンデマンドロードされる:

- エージェントハーネスや自律ワークフローを設計するとき
- 自律エージェントが静かに誤った原因をレビューするとき
- マルチエージェントパイプラインの信頼性を監査するとき

拡張版のパターン解説（実例・出典）はリポジトリの [`patterns/`](../../patterns/) ディレクトリにある。
